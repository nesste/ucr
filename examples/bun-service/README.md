# Bun Service Example

This example is a Bun-managed HTTP service target for UCR utilities, presets, and blocks.

The checked-in generated files show a working `posts` JSON API plus a `/health` endpoint composed from:

- `ts-runtime`
- shared utilities like `async-utility`, `object-utility`, and `validation-utility`
- shared presets like `service-preset` and `endpoint-preset`
- `entity-contract`
- `service-layer`
- `memory-repository`
- `input-validation`
- `health-route`
- `json-collection-route`
- `json-item-route`
- `bun-server`
