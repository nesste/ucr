# Next App Example

This example is a Bun-managed Next App Router target for UCR utilities, presets, and blocks.

The checked-in generated files show a full `posts` admin flow composed from:

- `ts-runtime`
- shared utilities plus Next-only UI utilities
- shared presets plus `form-preset` and `admin-page-preset`
- `entity-contract`
- `service-layer`
- `memory-repository`
- `input-validation`
- `api-client`
- `next-collection-route`
- `next-item-route`
- `data-table`
- `entity-form`
- `admin-page`
- `entity-detail-page`
